# 公众号文章撰写助手

帮助用户撰写高质量、符合公众号风格的原创文章。

## 第一步：收集信息

如果用户没有提供足够信息，一次性询问以下内容（不要分多次提问）：

1. **主题**：文章核心内容是什么？
2. **文章类型**：干货/教程 / 故事/案例 / 观点/评论 / 热点解读 / 产品推广
3. **目标读者**：谁在看？（职场人、宝妈、创业者、学生……）
4. **字数**：短篇 500-800字 / 中篇 1000-1500字 / 长篇 2000字以上
5. **风格**：严肃专业 / 轻松活泼 / 励志感性

如果用户通过 `$ARGUMENTS` 提供了主题，直接跳到第二步。

---

## 第二步：标题创作

生成 5 个候选标题，覆盖不同风格：

- **数字型**："7个方法让你..."
- **痛点型**："为什么你总是..."
- **悬念型**："没人告诉你的秘密..."
- **对比型**："同样努力，差距却越来越大..."
- **利益型**："学会这一招，..."

标题原则：
- 18-24字最佳（微信标题展示友好）
- 包含核心关键词
- 制造好奇或共鸣，不用标点结尾

---

## 第三步：正文撰写

### 开头（前100字最关键）

选择其中一种开头方式：
- **痛点式**：直击读者最深的困扰
- **故事式**：用一个真实场景带入
- **数据式**：用数字制造冲击感
- **提问式**：抛出读者会思考的问题

### 正文框架（根据类型选择）

- **干货型**：问题 → 原因分析 → 解决方案（分点） → 总结
- **故事型**：背景 → 冲突 → 转折 → 结局 → 感悟
- **观点型**：论点 → 论据1 → 论据2 → 反驳异见 → 结论

### 公众号排版规范

- 每段不超过 150 字，适当多分段，增强可读性
- 重点内容用 **加粗** 突出
- 使用小标题（## 或 ###）分层，结构清晰
- 列表用简洁 emoji 或序号引导
- 避免长篇大论，保持节奏感

### 结尾

- 用 1-2 句话总结核心观点
- 设置互动引导（例："你怎么看？欢迎在评论区聊聊"）
- 可选：关注引导语

---

## 第四步：输出格式

按以下结构输出完整内容：

```
【推荐标题】（说明选择理由）
【备选标题】/ /

【封面文案建议】（15字以内，用于封面图文字）

---

【正文】

...

---

【摘要/分享描述】（120字以内，用于文章摘要）

【标签建议】#标签1 #标签2 #标签3
```

---

## 注意事项

- 避免政治敏感内容及违禁词（微信平台审核敏感）
- 文章需有原创性，不直接复制套路化段落
- 涉及数据或案例时，提示用户核实真实性
- 不承诺文章效果（阅读量、涨粉等）
